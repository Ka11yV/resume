# Ka11y Git Resume

GitHub Pages로 배포할 수 있는 정적 이력서 사이트입니다.

## 로컬에서 확인하기

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 열면 됩니다.

## 수정 포인트

- 내용 수정: `resume-data.js`
- 구조 수정: `index.html`
- 디자인 수정: `styles.css`

## 배포

정적 파일만 사용하므로 GitHub Pages에서 브랜치 `main` / 루트(`/`)로 바로 배포할 수 있습니다.
